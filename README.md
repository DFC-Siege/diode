# Diode

Diode is a **Terminal User Interface (TUI)** application, written in **Rust**, that serves as a dual-pane file manager. It provides essential file manipulation capabilities, focusing on speed and efficiency for common file operations.

## ‚ú® Features

* **Dual-Pane File Management:** Navigate two directory structures simultaneously.
* **Quick Operations:** Quickly move or copy files and directories between the two panes.
* **Core File Features:** Perform basic file management actions, including:
    * Renaming files and directories.
    * Deleting files and directories.
    * Moving and copying files.
    * Creating new files and directories.
